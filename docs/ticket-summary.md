## トラブルシューティング

### よくある問題と解決策

**問題1**: Gemini APIのレート制限エラー
- **解決**: リクエスト間隔を空ける、エラーハンドリングを強化

**問題2**: Supabase RLSが機能しない
- **解決**: ポリシーを再確認、`auth.uid()`が正しく設定されているか確認

**問題3**: 外部レシピAPIのレスポンスが遅い
- **解決**: タイムアウト設定、`Promise.allSettled`でGemini APIの結果は先に返す

**問題4**: レスポンシブデザインの崩れ
- **解決**: Tailwindのブレークポイントを再確認、モバイルファーストで実装

**問題5**: ビルドエラー
- **解決**: `npm run build`でエラー確認、型エラーを修正
